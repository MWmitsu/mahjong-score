# 麻雀スコア（Web版）

ビルド不要・インストール不要の単一構成の Web アプリ。SwiftUI版から **Web に方針変更**したもの。
要件・点数計算・データモデル・確定判断はそのまま流用。

## 使い方（超かんたん）

- **PC**: `index.html` を**ダブルクリック**するだけでブラウザで開きます（Edge/Chrome/Firefox）。
- データはブラウザ内（`localStorage`）に保存。サーバー不要・オフライン動作。

### iPhone で使う（ホーム画面アプリ化）

PWA対応済み（manifest + Service Worker + アイコン）。HTTPSで配信すると、オフライン動作・全画面のアプリとして使えます。

**おすすめ：GitHub Pages（無料・gitがあればOK）**
1. GitHub アカウントを作成し、新しいリポジトリ（例 `mahjong-score`）を作る。
2. このフォルダで:
   ```
   git init
   git add .
   git commit -m "麻雀スコアアプリ"
   git branch -M main
   git remote add origin https://github.com/<ユーザー名>/mahjong-score.git
   git push -u origin main
   ```
3. GitHub のリポジトリ → Settings → Pages → Source を `main` / `/(root)` にして保存。
4. 数分後に `https://<ユーザー名>.github.io/mahjong-score/` が公開される。
5. iPhone の Safari でそのURLを開く →「共有」→「**ホーム画面に追加**」。アイコン付きアプリになります。

**その他**: Netlify / Vercel にフォルダをアップロードでも可（いずれも無料枠）。

> 注意: データは `localStorage` に**ブラウザ／端末ごと**保存。PCとiPhoneでデータは共有されません（端末間同期は将来の機能）。
> ローカルで `index.html` をダブルクリックして使うこともできますが、その場合 Service Worker は無効（オフラインPWA化はHTTPS配信時のみ）。

## 技術構成

- 素の HTML/CSS/JavaScript（**ES Modules不使用** = `file://` で直接動く）。
- 依存ライブラリ・ビルドステップなし。
- グローバル名前空間 `MJ` 配下に層を分離。

```
MahjongScoreWeb/
├── index.html
├── manifest.webmanifest
├── css/styles.css
└── js/
    ├── domain.js     列挙・既定ルール・点数計算・検証（純粋ロジック）
    ├── store.js      localStorage 保存層（IndexedDB/同期へ差し替え可能）
    ├── sampleData.js 初期ルール・サンプルデータ
    ├── selftest.js   点数計算の自己テスト（画面で検証）
    ├── ui.js         DOM補助・整形
    └── app.js        ルーター＋ホーム（screens は順次追加）
```

## 確定済みの設計判断

1. 箱下なし=精算粗点 `max(raw,0)`。卓合計は非ゼロサムになるが警告のみ。順位は raw で決定。
2. 1部屋=1種別（3麻/4麻）固定。
3. 成績・ランキングは常に種別を1つ選択（合算ビューなし）。
4. ルールに `pointToYenRate`（円/1pt）。設定時はチップ込み金額換算、未設定は分離表示。

その他: ルール編集は過去対局に遡及しない（結果はスナップショット保存）／削除は論理削除（`isDeleted`）／履歴ありの要素は無効化。

## 成績表方式の構成（現行）

- **部屋（成績表 / Session）** = 1日分など、複数の半荘を積む1枚の表。`js/screens/rooms.js`(一覧・新規作成) → `js/screens/sheet.js`(表本体)。
- 半荘の点数計算・合計・精算は `js/sheets.js`。
- データ: `sessions` コレクション。各 session に `rate`（円/pt）、`playerIds`（列）、`hanchans[]`（行＝各半荘の粗点と計算結果）、`chips{}`（最終行）。
- 精算: 各自 ＝ 合計ポイント × レート ＋ チップ枚数 × チップ単価。
