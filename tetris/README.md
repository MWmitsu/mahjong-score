# テトリス練習ツール

ぷよぷよテトリスのテトリス（ガイドライン／SRS）をベースにした、**練習特化**の単一構成 Web アプリ。
ビルド不要・依存なし。`tetris/index.html` をブラウザで開くだけで動きます（`file://` 直開きも可）。

## 主な機能

- **SRS 準拠の操作**：7-bag、ホールド、ゴースト、ソフト/ハードドロップ、壁蹴り（I ミノ含む）。
- **練習モード**
  - **フリー**：自由に積んで指慣らし・自由研究。
  - **テンプレ練習**：開幕テンプレ・PC（パーフェクトクリア）などを反復。
    - **ヒント**：次に置くべき目標位置を盤面に黄色で表示。
    - **正誤判定**：目標と違う置き方をすると警告。
    - **自動反復**：完了すると自動で最初からやり直し（設定でON/OFF）。
  - **掘り(Dig)**：お邪魔ブロックを掘って消す中盤・対応力の練習。
- **一手戻す（Undo）**：直前の操作（設置・ホールド）に戻れる。何手でも遡れる。
- **設定**：ゴースト／ヒント／重力（自動落下）／自動反復のON/OFF。
- **テンプレ検証**：全テンプレの整合性（盤外・重なり・穴・PC成立）を自動チェックして表示。

## 操作

| 操作 | キー | タッチ |
|---|---|---|
| 移動 | ← → | ← → |
| 右回転(CW) | ↑ または X | ⟳ |
| 左回転(CCW) | Z または Ctrl | ⟲ |
| 180°回転 | A | – |
| ソフトドロップ | ↓ | ↓ |
| ハードドロップ | Space | ハードドロップ |
| ホールド | Shift または C | ホールド |
| 一手戻す | U または Backspace | ↩戻す |
| リセット | R | リセットボタン |

DAS/ARR を内蔵しており、左右の長押しで連続移動します。

## テンプレの追加方法

`js/templates.js` の `list` に `steps` を追加するだけ。各手は
`{ piece, rot, col }` で表し、`col` は「そのミノが実際に占める一番左の列」（0=左壁）。
落下位置（行）はエンジンがその時点の積みから自動計算するため、
人間の操作「回転 → 列へ移動 → ハードドロップ」と一致します。

```js
{ id: "my_opener", name: "自作開幕", category: "開幕", pc: false,
  desc: "説明…",
  steps: [ { piece:"O", rot:0, col:0 }, { piece:"I", rot:1, col:9 } ] }
```

- `rot`: 0=出現 / 1=右回転 / 2=180 / 3=左回転
- `pc: true` にすると「全消し成立」を検証パネルでチェックします。
- 途中でラインが揃う構成は検証で警告されます（最終手で揃える前提）。

## 構成

```
tetris/
├── index.html
├── css/style.css
└── js/
    ├── engine.js     SRS・盤面・衝突・ライン消去（純粋ロジック）
    ├── templates.js  練習テンプレ定義（データ）
    └── app.js        描画・入力(DAS/ARR)・Undo・ヒント・各モード・検証
```
